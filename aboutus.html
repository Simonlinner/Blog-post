Syfte

Syftet med denna blogg hosting-sida är att den ska erbjuda en plattform för användare som vill skapa och publicera och hantera bloggar online. Det ska alltså vara relativt enkelt för i princip vem som helst att lägga upp ett blogginlägg och det ska även vara enkelt för andra att kunna läsa och lära sig. Alltså kommer det inte krävas någont konto för att man ska kunna läsa blogginläggen, dock skulle jag rekommendera om man skaffar ett konto för då så blir det enklare att spara all historik och såvidare för att användaren ska kunna få den bästa användarupplevelsen.

Hur har jag gjort

När jag började skapadet av min bloggsida började jag med att söka runt på google för att försöka hitta vilket ämne som jag skulle skriva om. Efter att ha letat ett tag så insåg jag att jag inte endast ville ha en blogg för mig själv utan jag ville helt enkelt skapa en sida där i princip vem som helst ska kunna lägga upp ett blogg-inlägg, och där i princip vem som helst ska kunna läsa inläggen. Såklart så började allt med att det var väldigt långtsamt arbete så jag i princip inte kunde någonting om hur man exempelvis flyttar runt föremål på ett snabbt och smidigt sätt, men desto mer man jobbade med det desto snabbare gick det då man kom ihåg vad de olika elementen gjorde för något. Utöver detta så har jag gjort mitt bästa för att ge användaren den bästa användarvänligheten genom att exempelvis göra så att knapparna i headern är interaktiva då de har hover animations så du vet när du håller över rubrikerna. Jag har även skapat inresseväckande intryck på första sidan genom att rekommendera de just nu trendande inläggen och under rubriken blogs i headern där man även kan se de mest sedda inläggen under den senaste tiden. Detta ska ha gjort hemsidan enkel och tydlig så att det går att ta sig runt på alla olika delar av sidan. Allt detta har jag gjort med hjälp av att använda css så att olika element får sin egen plats. Sidan är även ganska minimalistisk med en klassisk look som gör att den passar in för de flesta åldrarna, den passar till och med in på färgblinda då den inte blandar några primärfärger.

CSS Grid och CSS Flexbox

CSS Flexbox är en en-dimensionell layoutmodell som hanterar element i en rad eller i en kolumn. Brukar främst användas för centrering och enkel layoutanpassning. CSS Grid är en tvådimensionell layoutmodell som hanterar element i både rader och kolumner till skillnad från CSS FlexBox som endast kan hantera element i en rad eller i en kolumn. CSS Grid är alltså det bästa alternativet mellan dessa två när det gäller att skapa komplexa och strukturerade layouter. Både CSS Flexbox och CSS Grid är kraftfulla layouttekniker i webbutveckling, valet mellan dem beror på de specifika målen som du har för din layout. Flexbox passar oftast när du behöver hantera din layout inom en dimension, medans Grid är bäst när det gäller tvådimensionell layout.
